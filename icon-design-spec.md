# 🎨 API Navigator 图标设计规格

## 🏆 最终推荐方案：指南针+API设计

### 核心设计元素

#### 1. 主体：指南针圆环
外圆环：
- 直径: 100px (在128px画布中)
- 颜色: #007ACC (VSCode蓝)
- 线宽: 8px
- 样式: 实心圆环

内部指针：
- 颜色: #FF6B35 (橙红色，突出重点)
- 长度: 35px
- 指向: 右上角(45度)
- 尾部: 小三角形平衡

#### 2. 核心：API符号
大括号 { }：
- 位置: 指南针中心
- 颜色: #FFFFFF (白色)
- 字体: Monaco, 'Courier New' (等宽字体)
- 大小: 24px
- 样式: Bold

小点 (API端点)：
- 数量: 3个
- 颜色: #00D4AA (青绿色)
- 大小: 4px
- 位置: 括号内部垂直排列

#### 3. 背景和效果
背景：
- 颜色: 透明或白色
- 圆角: 16px (如果有背景色)

阴影效果：
- 模糊: 4px
- 偏移: 2px, 2px  
- 颜色: rgba(0,0,0,0.2)

## 🎨 配色方案

主配色：
--primary-blue: #007ACC     (VSCode主题蓝)
--accent-orange: #FF6B35    (指针橙红)
--success-green: #00D4AA    (API端点绿)
--neutral-white: #FFFFFF    (文字白色)
--shadow-gray: rgba(0,0,0,0.2) (阴影)
