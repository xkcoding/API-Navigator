# 产品上下文

## 产品愿景
为 Java/Kotlin 开发者提供一个强大且直观的 IntelliJ IDEA 插件，简化 REST API 的开发和维护过程。

## 目标用户
- Java/Kotlin 后端开发者
- 使用 Spring Boot、JAX-RS、Micronaut 框架的开发团队
- 需要快速导航 API 端点的开发者

## 核心价值主张
1. **快速导航**: 一键跳转到 API 端点定义
2. **多框架支持**: 支持主流 Java Web 框架
3. **智能提取**: 自动识别路径参数和映射注解
4. **开发效率**: 减少在大型项目中查找 API 的时间

## 竞争优势
- 专注于 Java 生态系统
- 深度集成 IntelliJ IDEA
- 支持复杂的注解模式
- 轻量级且性能优化

## 用户痛点
- 在大型项目中难以快速定位 API 端点
- 跨多个文件查找相关 API 映射
- 手动管理和文档化 API 路径
- 缺乏统一的 API 概览视图
